Extra species bundles.

Each subfolder is one species: a self-contained kit with the species
spec (.json) and its five text pools (names_female, names_male,
descriptions, pet_responses, disabilities). Drop any new species you
want to ship in here as a new subfolder; the in-game Mods menu will
list it for installation.

Players don't have to touch this folder — Mods -> Add or remove
extra species handles install/uninstall through a list dialog. This
folder exists to store extras that aren't currently active in
user_data/, plus to give modders a clean place to drop new bundles.
